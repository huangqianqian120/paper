# 组织激荡：咖啡指数与大模型时代的人才周期

> 2026年，一篇关于AGI时代人才流动与科技裁员的社科研究

## 摘要

本文以大模型团队负责人离职事件为核心案例，结合全球科技公司大规模裁员的行业背景，提出"技术封建主义"分析框架。研究发现：AGI时代的核心技术人才呈现"领地"式迁移特征，咖啡消耗量成为衡量组织压力的先行指标。

## 核心概念

- **咖啡指数**：科技园区咖啡消耗量，衡量程序员加班强度与组织压力的先行指标
- **技术封建主义**：AGI时代核心技术人才追求"技术领地"控制权的新型组织形态
- **双重挤压**：顶端人才被高薪挖走 + 基础程序员面临裁员风险

## 主要发现

1. 顶级AI人才从"雇佣关系"转向"技术领地"诉求
2. 开源vs闭源路线张力构成深层矛盾
3. 裁员表面降本，实则加速人才流失
